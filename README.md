# Sentiment Analysis using NLP

This project performs sentiment analysis on textual data such as reviews or tweets using Natural Language Processing (NLP).

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-Learn
- Matplotlib
- Seaborn

## Project Steps
1. Data Collection
2. Data Preprocessing
3. Feature Extraction using TF-IDF
4. Model Training using Logistic Regression
5. Model Evaluation
6. Visualization

## Output
The model predicts whether a review is Positive or Negative.
